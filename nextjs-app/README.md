# Mama Creole — marketing site

Next.js (App Router, TypeScript, Tailwind) marketing site for Mama Creole, 1029 Stuyvesant Ave., Union, NJ 07083.

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
```

**Vercel:** set the project Root Directory to `nextjs-app`; build command is `npm run build`.
